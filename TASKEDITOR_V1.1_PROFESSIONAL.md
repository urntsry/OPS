## 🎨 Win95 專業風格特點

### **視覺原則**
1. **無裝飾圖示** - 僅使用文字和符號
2. **工整對齊** - 所有元素精確對齊
3. **灰階色系** - 使用 Win95 標準色票
4. **3D 導角效果** - Inset/Outset 邊框
5. **固定字體** - 11px monospace

### **色票規範**
```css
背景灰: #C0C0C0
深灰邊框: #808080
黑色邊框: #000000
白色高光: #FFFFFF
選中藍: #000080
白色輸入框: #FFFFFF
```

## 🎯 設計哲學

> "專業工具應該像機台操作面板，不是遊戲介面"

**核心原則**：
1. **功能至上** - 每個元素都有明確用途
2. **視覺克制** - 避免不必要的裝飾
3. **工業美學** - 厚重、可靠、精確
4. **操作直覺** - 無需學習即可上手
