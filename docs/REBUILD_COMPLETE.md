## 🎨 設計系統規範

### 字體
- **主要**: MS Sans Serif / Segoe UI (11px)
- **數據**: Courier New (11px)
- **統一大小**: 11px

### 顏色
- **Win95 基礎**: #C0C0C0 (灰), #FFFFFF (白), #000000 (黑)
- **DOS 重點**: #00FF00 (綠), #FFAA00 (黃), #FF0000 (紅)
- **狀態顏色**: 只用於功能標示

### 間距
- **極小**: 2px
- **小**: 4px
- **中**: 8px
- **大**: 16px

### 元件
- **按鈕**: 2px 邊框 + 3D 導角
- **卡片**: 2px 邊框 + 白色背景
- **表格**: 1px 邊框 + 緊湊間距
