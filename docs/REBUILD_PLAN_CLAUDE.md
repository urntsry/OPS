### 元件設計

#### 按鈕
```
不要：花俏的漸變、陰影、動畫
要：  清晰的邊框、統一的高度、明確的狀態

[完成任務]  ← 實心、有重量
[取消]      ← 空心、較輕
```

#### 卡片
```
不要：圓角過大、陰影過重、顏色過多
要：  1px 邊框、8px 圓角、單一背景色
```
