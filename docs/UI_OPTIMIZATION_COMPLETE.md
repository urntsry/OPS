## 🎯 使用者體驗提升

### 場景 1：新增事項
```
使用者點擊行事曆 11/25 的格子
  ↓
彈出新增事項視窗
  ↓
選擇類型：交辦事項
輸入標題：整理316文件
  ↓
儲存 → 自動顯示在行事曆和交辦事項列表
```

### 場景 2：查看公告
```
使用者看到公告欄「[NEW] 本週五消防演練通知」
  ↓
點擊該公告
  ↓
彈出詳細視窗，顯示完整內容
  ↓
可查看連結、下載檔案
  ↓
點擊關閉或背景關閉視窗
```

### 場景 3：完成任務
```
使用者看到例行公事「週四丟垃圾」
  ↓
點擊 [ ] 勾選框 → 變成 [V]
  ↓
任務標記為完成
  ↓
積分自動增加
```
